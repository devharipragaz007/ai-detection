# AI Text Detector

Detect AI-generated content instantly. Paste any text and get a probability score, highlighted suspicious sentences, and a side-by-side diff view powered by Groq's LLM inference.

**Live Demo:** [ai-detection-iota.vercel.app](https://ai-detection-iota.vercel.app)

---

## What It Does

Content creators and writers need to know whether a piece of text was written by a human or generated by an AI. This tool solves that without requiring an account, an API key, or any setup.

Paste your text. Get results in seconds.

**Detection output includes:**
- AI probability score (0 to 100%)
- Highlighted suspicious sentences with per-sentence confidence
- Side-by-side diff view comparing original text against AI patterns
- Natural language explanation of why the content was flagged

**Humanize output:**
- One-click rewrite of AI-detected content into natural human writing
- Preserves original meaning while reducing AI pattern signatures
- Rewritten text can be re-analyzed immediately to verify the score improved

---

## How It Works

**Detection — LLM as Judge**

Text is sent to `llama-3.3-70b` via the Groq SDK through `/api/analyze`. The model acts as a judge, scoring the content and returning a structured explanation. A custom highlight engine maps the model's output back to individual sentences in the original text. Usage is tracked via PostHog analytics.

No embeddings. No vector DB. Fast, stateless inference on every request.

**Humanize — LLM Rewrite**

After detection, users can click **Rewrite to Humanize**. A separate `/api/rewrite` endpoint sends the flagged content back to the LLM with a different prompt, rewriting it to read as natural human writing while preserving the original meaning. The rewritten output can then be re-analyzed to verify the score improved.

## Running Locally

**Prerequisites:** Node.js 18+, a Groq API key

```bash
git clone https://github.com/devharipragaz007/ai-detector
cd ai-detector
npm install
```

Create a `.env.local` file:

```env
GROQ_API_KEY=your_groq_api_key_here
NEXT_PUBLIC_POSTHOG_KEY=your_posthog_key_here
NEXT_PUBLIC_POSTHOG_HOST=https://app.posthog.com
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
